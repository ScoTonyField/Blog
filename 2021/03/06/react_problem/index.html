React -- node_modules/canvans pig-config no found

问题描述

按照react的官方文档，npm >= 5.2的版本可以直接用npx create-react-app my-app代替npm install create-react-app && create-react-app my-app

一开始安装到node_modules/canvas时报错显示pkg-config no found

解决方法：brew install pkg-config

随后在文章中发现该文件是属于pango包。
